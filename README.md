A weather app written in Javascript. Utilize Web Components.

## Prerequisites
- [pnpm](https://pnpm.io/)

## Run
```bash
$ git clone git@github.com:VulpesVulpes825/weatherapp.git
$ cd weatherapp
$ pnpm install
$ pnpm run start
```

## Deploy
```bash
$ git clone git@github.com:VulpesVulpes825/weatherapp.git
$ cd weatherapp
$ pnpm install
$ pnpm run build
```
